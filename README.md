The Disassembler — Minecraft 1.20.1 (Forge)

Продвинутая система деконструкции предметов для Minecraft 1.20.1. Мод автоматически анализирует все доступные рецепты верстака и позволяет возвращать ресурсы с учетом качества оборудования.

⚙️ Логика работы и Баланс

Мод работает по принципу обратного инжиниринга, используя динамический анализ игровых рецептов.
1. Анализ рецептов 

   Мод автоматически подтягивает все возможные крафты, которые делаются на верстаке. Если предмет можно скрафтить — его можно разобрать (если он не в списке исключений).

2. Этапы обработки

   Фильтрация: Проверка предмета на наличие в списке исключений (blacklist).

   Парсинг: Поиск соответствующего рецепта в RecipeType.CRAFTING.

   Расчет выхода ресурсов:

        Базовый разборщик: Шанс возврата ресурсов — 75%.

        Улучшенный разборщик: Базовый шанс + Бонус установленной Пилы.

        Продвинутый разборщик: Базовый шанс + Бонус Пилы + 5% (Бонус Тира).

3. Гибкая настройка

   Конфиг-файлы: Позволяют управлять исключениями.

   Datapacks: Поддержка добавления кастомных рецептов разбора через JSON, если стандартный парсинг верстака не подходит для конкретного предмета.
